Cheat sheet

Variables

Contient une information dupliquée dans la feuille de styles.

$highlight: rgba(255, 255, 0, 0.35);
$header-height: 125px;

em {
  background: $highlight;
}

header {
  height: $header-height;
}

Résultat :

em {
  background: rgba(255, 255, 0, 0.35);
}

header {
  height: 125px;
}

Imbrication

Concevoir une hiérarchie de contenus.

ul {
  margin: 0;
  padding: 0;
    li {  
      list-style: none;
      display: block;
      a {
        text-decoration: none;
        margin: 0;
        padding: 10px;
        color:#888;
        line-height:30px;
      }
      &:hover {background:#eee;}
    }
  }
}

Résultat :

ul {
  margin: 0;
  padding: 0;
}
ul li {
  list-style: none;
  display: block;
}
ul li a {
  text-decoration: none;
  padding:10px;
  color:#888;
  display:block;
  line-height:30px;
}
ul li a:hover {
  background:#eee;
}

Décliner des propriétés :

aside{
  > span{
    border: {
      style: solid;
      color: #ccc;
        top: {
          width: 1px;
        }
        right: {
          width: 0px;
        }
        bottom: {
          width: 1px;
        }
        left: {
          width: 1px;
        }
    display: block;
    }
  }
}

Résultat :

aside > span{
  border-top: solid #ccc 1px;
  border-right: solid #ccc 0px;
  border-bottom: solid #ccc 1px;
  border-left: solid #ccc 1px;
  display: block;
}

Arithmétique

Utiliser des opérations mathématiques pour résoudre les calculs complexes ou à données variables.

$header-height: 125px;

aside {
  position: fixed;
  top: ($header-height + 60);
}

section {
  float: right;
  width: pourcentage(2/3);
}

Résultat :

aside {
  position: fixed;
  top: 185px;
}

section {
  float: right;
  width: 33.33333333%;
}

Mixins

Utiliser des fonctions.

@mixin dark-bkg {
  background:#111;
  color:#fff;
}

header{ @include dark-bkg; }

footer{ @include dark-bkg; }

Résultat :

header {
  background:#111;
  color:#fff;
}

footer {
  background:#111;
  color:#fff;
}

Paramètres

Rendre les fonctions modulables.

@mixin custom-bkg($bkg, $color) {
  background: $bkg;
  color: $color;
}

header{ @include custom-box(#fff, #000); }
aside{ @include custom-box(#eee, #333); }

Résultat :

header {
  background: #fff;
  color: #000;
}

footer {
  background: #eee;
  color: #333;
}

Extends

Créer des extensions de codes.

.error {
  border: 1px #F00;
  background: #FDD;
}

.badError {
  @extend .error;
  border-width: 3px;
}

Résultat :

.error, .badError {
  border: 1px #F00;
  background: #FDD;
}

.badError {
  border-width: 3px;
}

Import

Répartir et diviser le développement

@import 'partials/_vars';

body {
  color: $color;
}

/* content of _vars.scss
$color: #333;  */

Résultat :

body {
  color: #333;
}
